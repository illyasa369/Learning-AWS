# AWS CLI commands

## Install AWS CLI here:
https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html

Verify installation with:
```
aws --version
```
<br>

## Access Key command.
```
aws configure   # Begins credentials set up process.

aws sts get-caller-identity	# Returns information about the AWS identity currently in use.
```
<br>

## IAM commands:
```
aws iam list-users	# List all users in the account.
```
