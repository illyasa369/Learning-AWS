# AWS Storage

## EBS Volumes
- EBS (Elastic Block Store) Volume is a network drive that can be attached to instances.
- Allows EC2 instances to persist data even after an instance's termination.
- Bound to specific AZ.
<br>

## Amazon EFS
- Managed network file system that can be mounted to many EC2 instances.
- Can be used by many EC2 instances at the same time.
- Highly available - Data is copied across multiple AZs.
- Scalable - Scales automatically as data grows.
