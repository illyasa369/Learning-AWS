# Serverless
Serverless means you deploy code or services without managing servers; AWS handles infrastructure.
<br>

### AWS Lambda
- AWS's serverless compute service.
- Virtual functions - no servers to manage.
- Limited by time - Lambda functions have a max runtime of 15 mins.
- Run on-demand.
- Scaling is automated.

### Benefits of Lambda
- Easy pricing, pay per request and compute time.
- Integration with many AWS services and programming languages.
