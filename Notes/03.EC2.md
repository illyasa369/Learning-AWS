# AWS EC2
EC2 (Elastic Compute Cloud) is an Infrastructure as a Service offering that provides virtual servers in the cloud.

Capable of:
- Renting EC2 instances.
- Storing data on virtual drives.
- Load balancing across machines.
- Scaling services using Auto-Scaling Groups.
<br>

## EC2 Sizing and Configuration Options
- OS
- CPU
- RAM
- Storage space
- Network card
- Firewall rules
- Bootstrap script (EC2 user data)
<br>

## EC2 User Data
- Script ran once at the instance's first start.
- Used to automate boot tasks.
- User Data script runs with root user privileges.
<br>

## EC2 Instance types
- General Purpose
- Compute Optimized
- Storage Optimized
- Memory Optimized
- Accelerated Computing
- HPC Optimized

## EC2 Instances Purchasing Options
- On-Demand Instances: Pay as you go, most expensive.
- Reserved Instances: Commit for a longer period (1 & 3 years), 72% cheaper than On-Demand.
- Savings Plan: Commit to an amount of usage for a longer period (1 & 3 years).
- Spot instances: Unused capacity in an AZ, which can be lost if the spot price rises above the max price.
- Dedicated Hosts: Rent the entire physical server, giving control over instance placement.
- Dedicated Instances: Hardware dedicated to you however no control over instance placement.
- Capacity Reservations: Reserve capacity in a specific AZ for any duration.
